## Screenshot Guidelines

### Resolution
- Desktop: 1920x1080 or 1440x900
- Tablet: 768x1024
- Mobile: 375x812 (iPhone X/11/12)

### Format
- Use PNG format for better quality
- Optimize file size (keep under 500KB per image)
- Use descriptive filenames

### Content
- Show real product data
- Include UI interactions (hover states, dropdowns)
- Capture both light and dark themes
- Show responsive layouts

### Tools
- Browser DevTools for responsive views
- Screenshot extensions (Awesome Screenshot, Nimbus)
- Native OS screenshot tools
- Video to GIF converters for animations
