# Little Boys E-commerce

E-commerce application for Little Boys.

## Project Setup

This project uses Vite + React + TypeScript with Tailwind CSS and shadcn/ui components.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on http://localhost:8080

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Testing

```bash
npm run test        # Run tests once
npm run test:watch  # Run tests in watch mode
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- React Query
- React Hook Form
- Zod
- Vitest (Testing)

## Features

- Modern e-commerce interface
- Responsive design
- Product catalog
- Shopping cart
- Form validation
- Dark mode support
- Unit testing with Vitest
