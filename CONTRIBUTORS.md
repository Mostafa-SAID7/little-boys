## How to Become a Contributor

We welcome contributions from the community! Here's how you can help:

1. **Code Contributions**
   - Fix bugs
   - Add new features
   - Improve documentation
   - Optimize performance

2. **Non-Code Contributions**
   - Report bugs
   - Suggest features
   - Improve documentation
   - Help with translations
   - Share the project

## Recognition

Contributors will be recognized in the following ways:
- Listed in this file
- Mentioned in release notes
- GitHub contributor badge
- Special thanks in project README

## Getting Started

1. Read our [Contributing Guidelines](docs/CONTRIBUTING.md)
2. Check out [open issues](https://github.com/yourusername/little-boys/issues)
3. Join discussions in pull requests
4. Submit your first contribution!
